# Slide: Self-Evolving Local AI Butler

Slide is a voice-activated virtual assistant designed for local hardware execution, prioritizing privacy and low-latency performance.

### Key Features
* **Local Inference:** Powered by Ollama (Llama 3/Phi-4).
* **Recursive Logic:** Capability to modify its own scripts via verbal commands.
* **Computer Vision:** Biometric authentication using OpenCV.
